## 1. 地圖系統

### 1.1 地圖版本

玩家在生成地圖時可選擇版本。版本影響格子總數、商店數量與傳送點配置。

| 版本 | 格子總數 | 商店數量 | 傳送點數量 |

| 標準版 | 90格 | 8個 | 2–3個 |
| 完整版 | 120格 | 12個 | 6–8個 |

---

### 1.2 地形區域

地圖採直線移動設計，從起點出發走向終點，途經六大地形區域：

1. **新手村周邊** — 初始安全區域，敵人較弱，商店等級低
2. **森林** — 中低等威脅，開始出現中級商店
3. **沼澤** — 環境複雜，中級敵人為主
4. **雪山** — 高危險度，高等商店比例上升，冰系敵人
5. **火山** — 高危險度，傳說商店開始出現，火系敵人
6. **最終要塞** — 終局區域，史詩Boss、傳說商店、最終Boss

每個地形區域影響：商店等級分布機率、Boss等級、掉落品質。

### 1.5 白天與晚上機制

#### 時間循環（固定不變）
```
白天回合（所有玩家走一輪）
    ↓
晚上回合（所有玩家再走一輪）
    ↓
以上合計 = 1天，循環重複
```

- 白天與晚上各讓所有存活玩家行動一輪
- 白天＋晚上合計為「1天」
- 循環固定：白天→晚上→白天→晚上

#### 強制時間切換（道具或卡片觸發）
- 切換只影響當輪剩餘尚未行動的玩家
- 當輪結束後自動恢復正常循環
- 切換後不會永久改變時間，僅本輪有效

### 1.6 傳送格系統

### 2.3 神秘商店（召喚型）

召喚後開啟特殊商店，每個價格層級各隨機陳列5件商品，共30件。

**特性：**
- 商品從完整商品池隨機抽取，不受SL限制
- 僅召喚者可使用，其他玩家無法購買
- 使用後可正常移動，繼續本回合行動

**召喚方式：**

| 來源 | 費用 | 備註 |
|------|------|------|
| 機會卡：神秘商人的拜訪 | 免費 | 抽到即可使用 |
| 機會卡：神燈精靈的恩賜 | 免費 | 本次購買享八折優惠 |
| 道具：神秘商人契約書 | $20 | 商店購買 |
| 道具：神燈 | $30 | 限用3次，每次本次購買享八折 |

### 2.4 死亡商店

任何玩家永久死亡後，其持有的所有道具與金幣自動進入死亡商店。

**觸發時機：**
玩家走到商店格，或任何玩家召喚神秘商人時，若死亡商店有商品，自動附加死亡商店區塊於商店介面下方。

**死亡商店定價（原價×溢價）：**

| 原本層級 | 原價 | 死亡商店售價 | 計算方式 |
|--------|------|----------|--------|
| 初級 | $5 | $8 | 跳升一個層級 |
| 中級 | $8 | $12 | ×1.5 |
| 中高級 | $10 | $15 | 跳升一個層級 |
| 高級 | $15 | $25 | 跳升一個層級 |
| 稀有 | $20–$30 | $30–$45 | ×1.5 |
| 傳說 | $30–$50 | $45–$75 | ×1.5 |

**機會卡／命運卡的處理方式：**

第一步：拍賣競標
- 剩餘存活玩家依回合順序各出價一次（只出一次）
- 出價最高者得標，立即支付金幣

第二步：無人競標則進入死亡商店
- 價格由系統自動隨機決定，前端顯示範圍如下：

| 價格區間 | 金額範圍 | 前端顯示（D20）|
|--------|--------|------------|
| 低價 | $1–$10 | 擲出1到8 |
| 中價 | $11–$20 | 擲出9到15 |
| 高價 | $21–$30 | 擲出16到20（機率較高）|

## 3. 戰鬥系統

### 3.1 基本規則

- 玩家必須**完全消滅敵人**後，戰鬥才算結束
- 戰鬥結束後才輪到下一位玩家行動（戰鬥回合與移動回合是獨立的）
- 戰鬥回合與白天／晚上時間進展回合**完全獨立**，不互相影響
- 戰鬥中使用道具回復**消耗一個戰鬥回合**（戰士職業除外，其特性允許同回合攻擊與回復）

---

### 3.2 敵人標籤系統

只有帶有特殊標籤的敵人，才會在戰鬥開始時觸發D20強弱判定。無標籤敵人數值固定，不受時段影響。

| 標籤 | 白天判定（D20）| 晚上判定（D20）|
|------|-----------|-----------|
| 【晝強】 | 擲出14–20：敵人強化；其餘正常 | 不觸發，正常數值 |
| 【夜強】 | 不觸發，正常數值 | 擲出11–20：敵人強化；其餘正常 |
| 【混沌】 | 擲出1–10：弱化；11–20：強化 | 擲出1–10：弱化；11–20：強化 |
| 【晝夜】 | 擲出1–10：強化；11–20：弱化 | 擲出11–20：強化；1–10：弱化 |

**強化效果示例：** 攻擊+30%、HP+20%、防禦+2  
**弱化效果示例：** 攻擊-20%、HP-15%、防禦-1

> 前端統一以「擲出X到Y時觸發強化/弱化」方式顯示，不顯示百分比。

---

### 3.3 Boss戰輔助機制

#### 一般Boss戰（初級、中級、史詩級Boss）

主場玩家（踩到Boss格的玩家）可邀請其他玩家輔助。輔助玩家的傷害依下列比例折算後計入：

| 輔助行動類型 | 傷害計入比例 |
|----------|----------|
| 普通攻擊 | 25% |
| 技能攻擊 | 35% |
| 白魔法師回復 | 100%（視為等效傷害）|
| 道具使用造成的效果 | 50% |

#### 最終Boss戰（虛空之主）

- 所有玩家傷害**完全不打折**，一律全額計算
- 性質為各自獨立攻擊同一目標，無主從關係
- 「最後一擊」勝利機制保留（最後一擊玩家獲得額外獎勵）

### 3.6 死亡與復活機制

#### HP歸零時（瀕死狀態）

1. 玩家進入「**瀕死狀態**」，本回合起無法行動
2. 每回合輪到該玩家時：
   - 自動扣除最大HP的5%（瀕死流血效果）
   - 擲D20嘗試復活

#### 復活判定（D20）

| 擲出數字（前端顯示）| 結果 |
|---------------|------|
| 1–10：復活失敗 | 繼續瀕死，本回合扣血後等待下一回合 |
| 11–19：復活成功 | 恢復最大HP的20%，可繼續行動 |
| 20：完美復活 | 恢復最大HP的30%，本回合可立即行動 |

#### 鳳凰羽翼斗篷（自動觸發裝備）

- 持有此道具的玩家，HP歸零時自動觸發
- 直接恢復50% HP，無需擲骰，跳過瀕死狀態
- 觸發後斗篷消失，無法再次使用

#### 永久死亡條件

以下任一條件成立時，玩家**永久出局**：
- 連續3回合復活擲骰失敗
- 瀕死流血扣到HP歸零（二次歸零）

永久死亡後：
- 所有道具與金幣進入死亡商店
- 玩家出局，不再參與後續回合

## 4. 職業系統

**定位：** 高防禦、高HP、近戰主力，最適合前線抗傷。

| HP | 120 |
| 攻擊 | 2d8 |
| 防禦 | 10 |
| 速度 | 3 |

**特性一：防禦姿態**
每回合可宣告進入防禦姿態。效果：當回合所有傷害減半，但**無法進行任何攻擊**。防禦姿態與道具使用可並存。

**特性二：戰地急救**
使用回復道具時**不消耗戰鬥回合數**，可在同一回合內同時攻擊與使用回復道具。此特性為戰士獨有，其他職業使用道具需消耗一個回合。

**回復方式：** 主要依賴道具，無被動回復能力。

### 4.2 黑魔法師

**定位：** 高爆發魔法傷害為主，脆皮，具備少量自我回復，高風險高報酬。

**特性二：暗黑汲取**
每次魔法攻擊命中後，回復本次造成傷害的10%為自身HP。此效果**僅限自我回復，不可用於隊友**。

**特性三：魔力耗盡懲罰**
魔力歸零時，無法使用任何特性技能，只能進行普通攻擊。直到魔力恢復方可重新使用技能。

**魔力回復：** 每回合自動回復2點（戰鬥外）；魔力泉格子可回復6點。

## 8. 機制系統

### 8.1 陷阱系統

#### 基本規則

| 規則 | 說明 |
|------|------|
| 設置限制 | 每位玩家同時最多放置**2個陷阱** |
| 無效對象 | 陷阱對**設置者本人**無效，不會觸發 |
| 觸發效果 | 其他玩家踩中後觸發效果，支付金幣給設置者 |
| 消失規則 | 陷阱觸發**一次後消失**（除幽靈鎖鏈外持續多回合）|
| 隱藏性 | 陷阱位置**對其他玩家不可見** |
| 偵測方式 | 持有**探測礦石棒**（$10）可偵測前方1格是否有陷阱 |

#### 陷阱等級升降（命運卡觸發）

| 操作 | 效果 |
|------|------|
| 等級+1 | 陷阱效果提升至上一層級（如$5升為$8效果）|
| 等級-1 | 陷阱效果降至下一層級（如$15降為$10效果）|
| 等級+2 | 大幅提升，高難度判定 |
| 等級-2 | 大幅降低，高成功率判定 |
| 升降失敗 | 依卡片設計各有不同後果（陷阱消失、反彈等）|

> 所有判定以D20擲骰，前端顯示「擲出X到Y視為成功」方式呈現。

### 8.3 骰子系統

#### 可用骰子種類

D4、D6、D8、D10、D12、D20

#### 前端顯示規則（設計原則）

所有判定機制一律以**「擲出X到Y視為成功」**方式顯示給玩家，**不顯示百分比**。

範例：
- 不顯示「70%機率成功」
- 顯示「擲出1–20，擲出1到14視為成功」

#### 後端邏輯

後端使用隨機數字產生（`Math.random()` 或等效方法），骰子面數決定隨機範圍。系統自動計算：
- 商店陳列內容與特殊道具出現
- 寶箱等級與品質
- 隨機事件觸發結果
- 傳送判定、復活判定等

玩家不需手動計算任何機率。

### 8.4 勝利點數系統

#### 獲得方式

| 條件 | 勝利點數 |
|------|--------|
| 最終Boss最後一擊 | +3點 |
| 參與擊殺最終Boss（非最後一擊）| +1點 |
| 最先抵達最終Boss格 | +3點 |
| 擊殺史詩級Boss | +2點（每次）|
| 擊殺中級Boss | +1點（每次）|
| 成功觸發陷阱讓其他玩家中招 | +1點（每次）|
| 持有最多金幣者（遊戲結算時）| +2點 |
| 持有最多道具者（遊戲結算時）| +1點 |
| 抽到「終極試煉」命運卡並在試煉中勝利 | +2點 |

#### 遊戲結束條件

以下任一條件達成時，遊戲立即結算：

1. **最終Boss被擊敗** — 最後一擊的玩家獲得主要勝利獎勵
2. **所有其他玩家全部永久死亡** — 最後存活的玩家勝利
3. **玩家累積到足夠勝利點數**（如有設定目標分數時）

累積**最多勝利點數**的玩家為最終勝利者。

### 8.5 無PVP設計原則

玩家之間**沒有直接戰鬥**。所有玩家互動透過以下**間接機制**進行：

| 機制 | 說明 |
|------|------|
| 陷阱系統 | 秘密設置格子，等待其他玩家踩中 |
| 詛咒之石 | 主動對他人道具施加詛咒 |
| 標籤刻印石 | 對特定格子施加標籤，影響他人遭遇的敵人 |

| 時間操控道具 | 強制切換時間影響其他玩家的戰鬥條件 |
| 死亡商店競標 | 當玩家死亡後，競標其道具 |
| 最終Boss最後一擊競爭 | 最後一擊獎勵豐厚，玩家會為此競爭 |

### 9.3 次元口袋詳細規則

- **時限：** 持有後滿5天（10回合）觸發消失判定
- **道具編號：** 依放入順序編號（1號、2號、3號…）
- **編號永久性：** 取出道具後，剩餘道具的編號**不重新排列**

#### 消失判定流程

每天（白天開始時）觸發一次：

1. 先擲D20
2. 若道具數量≤20：擲出的數字對應編號的道具消失
3. 若道具數量＞20且D20擲出20：追加下一顆骰子（依下表）
4. 若擲出數字＞目前道具總數：消失最後一個編號的道具

| 口袋內道具數量 | 追加骰子 |
|------------|--------|
| 1–20件 | 僅D20，不追加 |
| 21–24件 | D20擲出20後，追加D4 |
| 25–26件 | D20擲出20後，追加D6 |
| 27–28件 | D20擲出20後，追加D8 |

### 9.4 回合結構詳細說明

```
【一天的結構】
白天回合：
  玩家1行動（移動、事件、戰鬥等）
  玩家2行動
  玩家3行動（若有）
  玩家4行動（若有）

夜晚回合：
  玩家1行動
  玩家2行動
  玩家3行動（若有）
  玩家4行動（若有）

以上合計 = 第N天結束，進入第N+1天白天
```

每位玩家每回合的完整行動包含：
1. 檢查跳過回合狀態（若有）
2. 若瀕死：自動扣血 + 擲D20嘗試復活
3. 擲骰決定移動步數（通常D6）
4. 移動至新格子，觸發格子事件
5. 處理所有事件效果（戰鬥、商店、卡片等）
6. 結算本回合結尾效果（法師MP回復等）
7. 宣告結束回合，切換至下一位玩家

### 9.5 死亡機制完整說明

#### 瀕死狀態（HP歸零觸發）

進入瀕死狀態後，每回合發生：

| 步驟 | 說明 |
|------|------|
| 1. 自動扣血 | 扣除最大HP的5%（無法透過技能或道具抵消，除非使用瀕死道具）|
| 2. 擲D20 | 嘗試復活（門檻見下表）|
| 3. 記錄失敗次數 | 連續3次失敗→永久死亡 |

#### 復活判定表

| 前端顯示結果 | 擲出數字 | 效果 |
|----------|--------|------|
| 復活失敗 | 1–10 | 繼續瀕死，本回合扣血後等待下一回合 |
| 復活成功 | 11–19 | 恢復最大HP的20%，可繼續行動 |
| 完美復活 | 20 | 恢復最大HP的30%，本回合可立即行動 |
| 使用死亡抗拒符 | 6以上 | 門檻降低，擲出6以上即視為成功 |

#### 永久死亡後處理

1. 玩家**出局**，不再參與任何回合
2. 持有的所有道具進入**死亡商店**（按價格溢價販售）
3. 持有的金幣進入**死亡商店**作為購買資金池（或依規則分配）
4. 持有的機會卡/命運卡進入**拍賣**，無人競標後進入死亡商店
